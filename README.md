# Organizador de Arquivos em Python

Projeto em Python que organiza arquivos automaticamente em pastas com base na extensão.

## Funcionalidades
- Lê arquivos de uma pasta de origem
- Identifica a extensão de cada arquivo
- Cria pastas automaticamente
- Move os arquivos para a categoria correta

## Categorias
- imagens
- pdfs
- documentos
- planilhas
- audios
- videos
- compactados
- outros

## Tecnologias
- Python
- os
- shutil

## Objetivo
Praticar lógica de programação, automação e manipulação de arquivos no sistema operacional.
